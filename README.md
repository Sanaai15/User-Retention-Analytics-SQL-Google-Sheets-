# User Retention Cohort Analysis

## Project Overview

This project focuses on user retention analysis using cohort analysis techniques. The objective was to evaluate user engagement over time and compare retention performance between promotional and organic acquisition channels.

## Tools Used

- SQL
- DBeaver
- Google Sheets

## Key Tasks

- Extracted and prepared user activity data from a relational database using SQL.
- Cleaned and transformed date fields stored in different formats.
- Calculated user cohorts based on first activity month.
- Computed user tenure and monthly retention metrics.
- Built interactive cohort tables in Google Sheets.
- Applied conditional formatting to visualize retention trends.
- Compared retention performance of promotional and organic user groups.

## Skills Demonstrated

- Cohort Analysis
- Retention Rate Calculation
- SQL Data Extraction and Transformation
- Date Parsing and Data Cleaning
- Data Aggregation and Filtering
- User Behavior Analysis
- Google Sheets Reporting and Visualization

## Key Findings

The analysis revealed differences in retention patterns between promotional and organic users, providing insights into acquisition quality and long-term user engagement.
